物理层

比特流传输与带宽、奈奎斯特/香农上限

考点速记

  • 四特性：机械、电气、功能、过程
  • 视角：时域与频域，频域更直观
  • 傅里叶：复杂周期信号可分解为多正弦叠加
  • 带宽：理论无限；有效带宽受物理信道衰减限制
  • 奈奎斯特（无噪声）：C = 2·H·log₂(V)
  • 香农（有噪声）：C = H·log₂(1 + S/N)；S/N = 10^{dB/10}
  • 综合：最大速率为两者的较小值
  • 传输方式：模拟/数字，串行/并行，单工/半双工/全双工

速算工具

奈奎斯特
C = 2·H·log₂(V)
香农
C = H·log₂(1 + S/N)
分贝换算
S/N = 10^{dB/10}
最终速率
C = min(C_Nyquist, C_Shannon)

传输介质

导引型（有线）

  • 双绞线：STP/UTP，类别越高带宽越大
  • 同轴电缆：抗干扰强于双绞线
  • 光纤：带宽极大、抗干扰强、距离远；成本高、易损

非导引型（无线）

  • 无线电波、微波（地面/卫星）、光波（红外/可见光）

蜂窝网络

  • 六边形小区+基站，频率复用
  • 相邻小区频率不同，每小区最多可用频率约为 N/3

传输方式与设备

  • 基带传输：数字信号直传，使用 Codec
  • 频带传输：数字调制为模拟信号，使用 Modem
  • 物理层设备：中继器、集线器

编码与调制

基带编码

  • NRZ：连续比特难同步
  • 曼彻斯特：位中跳变自同步
  • 差分曼彻斯特：位开始是否跳变编码，抗极性反转

数字调制

  • ASK（幅移）、FSK（频移）、PSK（相移）
  • 混合调制：提高每码元比特数

多路复用

  • FDM：频分复用，含保护带
  • WDM：光纤上的频分（按波长）
  • TDM：时分固定时隙；STDM：按需分配时隙
  • CDMA：正交码片，接收端与码片序列内积解码
