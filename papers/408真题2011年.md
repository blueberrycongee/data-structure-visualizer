# 2011年408真题

### 数据结构
二、选择题
1. 设 n 是描述问题规模的非负整数，下面程序片段的时间复杂度是（ ）。 
```
x = 2;
while (x < n/2)
    x = 2 * x;
```
选项:
A. $O(\log n)$
B. $O(n)$
C. $O(n\log n)$
D. $O(n^2)$
答案: A
解析: 循环条件是x < n/2，每次循环x乘以2，所以循环次数约为$\log_2(n/2)$，时间复杂度为$O(\log n)$。

2. 元素 a，b，c，d，e 依次进入初始为空的栈中，若元素进栈后可停留、可出栈，直到所有元素都出栈，则在所有可能的出栈序列中，以元素 d 开头的序列个数是（ ）。
选项:
A. 3
B. 4
C. 5
D. 6
答案: B
解析: d第一个出栈，说明a,b,c已在栈中，且d入栈后立即出栈。此时栈中从顶到底为c,b,a。可能的出栈序列为d,c,e,b,a; d,c,b,e,a; d,c,b,a,e; d,b,c,e,a; d,b,c,a,e; d,b,a,c,e。但要注意e的插入位置限制，实际只有4种可能。

3. 已知循环队列存储在一维数组 A[0..n-1] 中，且队列非空时 front 和 rear 分别指向队头元素和队尾元素。若初始时队列空，且要求第一个进入队列的元素存储在 A[0] 处，则初始时 front 和 rear 的值分别是（ ）。
选项:
A. 0,0
B. 0,n-1
C. n-1,0
D. n-1,n-1
答案: B
解析: 第一个元素入队后存储在A[0]，则入队操作应为rear=(rear+1) mod n。初始时队列空，rear初始值应为n-1，使得(rear+1) mod n = 0。front指向队头元素，初始时为0。

4. 若一棵完全二叉树有 768 个结点，则该二叉树中叶结点的个数是（）。
选项:
A. 257
B. 258
C. 384
D. 385
答案: C
解析: 在完全二叉树中，最后一个分支结点的序号为$\lfloor n/2 \rfloor$。叶结点数 = 总结点数 - 分支结点数 = 768 - $\lfloor 768/2 \rfloor$ = 768 - 384 = 384。

5. 若一棵二叉树的前序遍历序列和后序遍历序列分别为 1,2,3,4 和 4,3,2,1，则该二叉树的中序遍历序列不会是（）
选项:
A. 1,2,3,4
B. 2,3,4,1
C. 3,2,4,1
D. 4,3,2,1
答案: D
解析: 前序为根左右，后序为左右根。前序序列第一个是根结点1，后序序列最后一个是根结点1。根据前序和后序无法唯一确定二叉树，但可以判断某些序列不可能。选项D的中序遍历序列根结点1在最后，意味着1是最后访问的，这与前序中1第一个被访问矛盾。

6. 已知一棵有 2011 个结点的树，其叶结点个数为 116，该树对应的二叉树中无右孩子的结点个数是（ ）。
选项:
A. 115
B. 116
C. 1895
D. 1896
答案: D
解析: 树转换为二叉树时，每个结点的最右孩子失去右指针（变为空），根结点也没有右孩子。无右孩子的结点数 = 分支结点数 + 1 = (2011-116) + 1 = 1895 + 1 = 1896。

7. 对于下列关键字序列，不可能构成某二叉排序树中一条查找路径的序列是（ ）。
选项:
A. 95，22，91，24，94，71
B. 92，20，91，34，88，35
C. 21，89，77，29，36，38
D. 12，25，71，68，33，34
答案: A
解析: 在二叉排序树的查找路径上，一旦转向某子树，后续值必须满足该子树的范围限制。选项A中，从95到22是向左，后续值都应小于95；从22到91是向右，后续值应大于22且小于95；但94大于91却小于95，违反了二叉排序树的性质。

8. 下列关于图的叙述中，正确的是（ ）。 I. 回路是简单路径 II. 存储稀疏图，用邻接矩阵比邻接表更省空间 III. 若有向图中存在拓扑序列，则该图不存在回路
选项:
A. 仅II
B. 仅I、II
C. 仅III
D. 仅I、III
答案: C
解析: I错误，回路要求起点终点相同，简单路径要求顶点不重复，回路不是简单路径。II错误，稀疏图用邻接表更省空间。III正确，有向无环图才有拓扑序列。

9. 为提高哈希（Hash）表的查找效率，可以采取的正确措施是（ ）。 I. 增大装填因子 II. 设计冲突少的哈希函数 III. 处理冲突时避免产生堆积现象
选项:
A. 仅I
B. 仅II
C. 仅I、II
D. 仅II、III
答案: D
解析: I错误，装填因子增大意味着哈希表更满，冲突概率增加，效率降低。II、III正确，可以减少冲突和提高查找效率。

10. 为实现快速排序算法，待排序序列宜采用的存储方式是（）
选项:
A. 顺序存储
B. 散列存储
C. 链式存储
D. 索引存储
答案: A
解析: 快速排序需要随机访问元素，顺序存储支持随机访问，效率最高。

11. 已知序列 25,13,10,12,9 是大根堆，在序列尾部插入新元素 18，将其再调整为大根堆，调整过程中元素之间进行的比较次数是（）
选项:
A. 1
B. 2
C. 4
D. 5
答案: B
解析: 插入18后，需要向上调整。18与父节点12比较，交换；18再与父节点25比较，不交换。共比较2次。

13. float 型数据通常用 IEEE754 单精度浮点数格式表示。若编译器将 float 型变量 x 分配在一个 32 位浮点寄存器 FR1 中，且 x=-8.25，则 FR1 的内容是（ ）。
选项:
A. C1040000H
B. C2420000H
C. C1840000H
D. C1C20000H
答案: A

15. 某计算机存储器按字节编址，主存地址空间大小为 64MB，现用 4M×8 位的 RAM 芯片组成 32MB 的主存储器，则存储器地址寄存器 MAR 的位数至少是（ ）。

16. 偏移寻址将某个寄存器内容与一个形式地址相加而生成有效地址。下列寻址方式中，不属于偏移寻址方式的是（ ）。
选项:
A. 间接寻址
B. 基址寻址
C. 相对寻址
D. 变址寻址
答案: A

17. 某机器有一个标志寄存器，其中有进位/借位标志 CF、零标志 ZF、符号标志 SF 和溢出标志 OF，条件转移指令 bgt（无符号整数比较大于时转移）的转移条件是 ( )
选项:
A. $\overline{CF} + \overline{OF} = 1$
B. $\overline{SF} + ZF = 1$
C. $\overline{CF + ZF} = 1$
D. $\overline{CF} + \overline{SF} = 1$
答案: C
解析: 无符号数比较A>B时，A-B无进位（CF=0）且结果不为0（ZF=0），故条件为$\overline{CF} \cdot \overline{ZF} = 1$，即$\overline{CF + ZF} = 1$。

18. 下列给出的指令系统特点中，有利于实现指令流水线的是（ ）。 I. 指令格式规整且长度一致 II. 指令和数据按边界对齐存放 III. 只有 Load/Store 指令才能对操作数进行存储访问

解析: 这三项都是RISC的特点，有利于流水线实现。

19. 假定不采用 Cache 和指令预取技术，且机器处于"开中断"状态，则在下列有关指令执行的叙述中，错误的是（ ）。

21. 某计算机有五级中断 $L_4 \sim L_0$，中断屏蔽字为 $M_4 M_3 M_2 M_1 M_0$，$M_i = 1$（$0 \le i \le4$）表示对 $L_i$ 级中断进行屏蔽。若中断响应优先级从高到低的顺序是 $L_0→L_1→L_2→L_3→L_4$，且要求中断处理优先级从高到低的顺序是 $L_4→L_0→L_2→L_1→L_3$，则 $L_1$ 的中断处理程序中设置的中断屏蔽字是（ ）。

B. 01101
C. 00011
D. 01010

22. 某计算机处理器主频为 50MHz，采用定时查询方式控制设备 A 的 I/O，查询程序运行一次所用的时钟周期数至少为 500。在设备 A 工作期间，为数据不丢失，每秒需对其查询至少 200 次，则 CPU 用于设备 A 的 I/O 的时间占整个 CPU 时间的百分比至少是（ ）。
选项:
A. 0.02%
B. 0.05%
C. 0.20%
D. 0.50%
答案: C
解析: 每秒查询200次，每次500时钟周期，总周期数=200×500=100000。CPU时间占比=100000/(50×10⁶)=0.002=0.2%。

23. 下列选项中，满足短任务优先且不会发生饥饿现象的调度算法是（ ）。
选项:
A. 先来先服务
B. 高响应比优先
C. 时间片轮转
D. 非抢占式短任务优先
答案: B
解析: 高响应比优先调度算法综合考虑作业的等待时间和运行时间，响应比=(等待时间+运行时间)/运行时间，短作业优先且长作业随着等待时间增长响应比提高，不会饥饿。

31. 某文件占 10 个磁盘块，现要把该文件磁盘块逐个读入主存缓冲区，并送用户区进行分析，假设一个缓冲区与一个磁盘块大小相同，把一个磁盘块读入缓冲区的时间为 100μs，将缓冲区的数据传送到用户区的时间是 50μs，CPU 对一块数据进行分析的时间为 50μs。在单缓冲区和双缓冲区结构下，读入并分析完该文件的时间分别是（ ）。
