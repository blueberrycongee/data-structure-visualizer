性能与度量

RTT、吞吐量、带宽、PPS、时延带宽积

基本度量

  • 带宽/比特率（bps）
  • 分组转发率（PPS）

考点速记

  • 效率题：层间开销固定时，按载荷与总开销计算效率
  • 时延比较：先列总时延公式，再建不等式求参数范围
  • 最优分组大小：对总时延对 p 求导，令导数为零求驻点
  • 最大速率：奈奎斯特与香农分别计算，取较小值
  • 分贝换算：S/N = 10^{dB/10}

时延构成

  • 传输时延：数据大小 / 带宽
  • 传播时延：距离 / 传播速度
  • 处理时延
  • 排队时延

其他度量

  • RTT
  • 吞吐量
  • 利用率
  • 丢包率
  • 时延带宽积

滑动窗口性能公式

  • 发送时延：T_s = 帧长度(bit) / 带宽(bps)
  • 发送周期：T_f = W_S · T_s
  • 利用率：U = T_f / (T_s + RTT + T_a)
  • 吞吐量：Throughput = U · 带宽
  • 确认开销：捎带确认可视为 T_a = T_s，否则按题目说明
